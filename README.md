# Crochet Haven – Handmade Crochet E-Commerce Website

Crochet Haven is a beautiful and fully responsive mini e-commerce website created for selling handmade crochet products. The project was developed using HTML, CSS, Bootstrap, and JavaScript as part of a Web Design final project.

## Features
- Responsive design for desktop, tablet, and mobile
- Beautiful handmade crochet aesthetic
- Dynamic product display using JavaScript
- Shopping cart functionality
- Product search and category filtering
- Dark mode toggle
- Checkout form with validation
- Bootstrap components and animations
- Accessible and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Pages Included
- Home Page
- About Page
- Products Page
- Cart / Checkout Page

## How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser
